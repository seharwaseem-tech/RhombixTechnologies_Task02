# RhombixTechnologies_Task02
A social media website using HTML, CSS, and JavaScript with light and dark modes.

# Project Overview
This project is a **social media Website** inspired by Facebook.  
It is built using **HTML, CSS, and JavaScript** and includes a **light and dark theme toggle** for better user experience.  

This project was created as part of my **internship task at Rhombix Technologies**.


# Features
- Social media style homepage layout  
- Light and dark theme toggle using JavaScript  
- Responsive and clean user interface  
- Organized sections for profile, posts, and navigation  

# Technologies Used
- HTML5  
- CSS3  
- JavaScript  

# Purpose of This Project
The purpose of this task is to demonstrate:  
- Understanding of HTML page structure  
- CSS styling, layouts, and responsiveness  
- JavaScript interactivity for theme switching  
- Frontend development skills inspired by real-world websites  

# Folder Structure
- `index.html` — Main HTML page  
- `style.css` — Stylesheet for layout and design  
- `script.js` — JavaScript for light/dark mode and interactivity  
- `images/` — Folder containing all image assets used in the website
